# 🌍 GlobalLingo Translator

GlobalLingo Translator is a modern web application that allows users to translate text into multiple languages instantly. The platform also includes speech recognition and text-to-speech features, enabling users to speak, translate, and listen to translated content seamlessly.

## ✨ Features

* 🌐 Translate text between multiple languages
* 🎤 Speech-to-Text functionality
* 🔊 Text-to-Speech support for translated text
* 🚀 Fast and accurate translations
* 📱 Responsive and user-friendly interface
* 🌎 Supports a wide range of global languages
* 💡 Easy-to-use and beginner-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Translation API
* Web Speech API (Speech Recognition)
* Text-to-Speech API
* Mongo DB

## 📸 How It Works

1. Enter text or use the microphone to speak.
2. Select the source and target languages.
3. Click the **Translate** button.
4. View the translated text instantly.
5. Click the **Listen** button to hear the translated text.

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/GlobalLingo-Translator.git
```

2. Navigate to the project folder:

```bash
cd GlobalLingo-Translator
```

3. Open `index.html` in your browser.
